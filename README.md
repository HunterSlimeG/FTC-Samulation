# Welcome to the FTC Sam-ulation!
The FTC Sam-ulation is a physics-based simulation developed in the Godot Engine. While the origin of the project was to help showcase [our](#about-us)  robot for the DECODE season at the Florida State Championship, it grew to be a tool used to inspire younger students and help develop the skills used in Tele-Op. 
## About Us
The FTC Sam-ulation was first developed by [Mario's Mechanics](https://www.instagram.com/19954_marios_mechanics/), team 19954, in early 2026. 
The name was coined in relation to the original developer, Sam.
